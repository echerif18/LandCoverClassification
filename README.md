# ThesisLandCover

The present project presents testing experiments for land cover classification on data from the Amazon Basin. The experiments comprises differents data sets' and models' configurations. The data set employed is a set of 256*256 tiles of stacked multi-spectral (Sentinel-2) and multi-temporal (Sentinel-1) images. A detailed description about the data format is described in the thesis document.
This project is developed with based on the NGC docker container 20.03-tf1-py3.
